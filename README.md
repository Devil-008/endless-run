# Flappy Bird - Nature Edition

A beautiful Flappy Bird-like game built with React and Vite, featuring a nature theme with realistic visuals.

## Features

- Smooth gameplay with realistic physics
- Nature-themed graphics and animations
- Score tracking and high score system
- Responsive design
- Collision detection
- Day/night sky gradient background

## Controls

- **Space Bar** or **Click/Tap** - Make the bird flap
- **Enter** - Start/Restart game

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
